RoadSignModal

This component will display a modal with an image of the chosen road sign. When the image, or the "i"-button is pressed, it will display the name and description related to said sign. To close the modal, either press the "X"-button or outside of the modal.

Properties:
Name Type Description
modalVisible object

If the modal is visible or not, plus functions to open and close it
selectedSign object

Contains name, description and image source of the chosen sign
selectedSignCode string

The sign code of the selected sign (example: 100_1)
