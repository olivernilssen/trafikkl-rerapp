DraggablesWithMenu

DraggablesWithMenu

This components collects all the draggable components into one. The components contains the list of draggables, the draggables object, the draggable menu and the dropzone area.

Properties:
Name Type Description
topMenuOpen boolean

If the topMenu is open or not in view
draggables array

List of all draggables in view
setDraggables function

Function to update the draggables array
actionList array

List of all actions taken (drawing or adding draggables)
setActionList function

Update the actionList (add or remove)
deletingItemId int

Integer to let the component know which draggable to delete
name string

Name of the drawing view ('veikryss'... etc)
extensionType string

Information about which extension is being used ("gangfelt", "sykkelfelt", etc)
setExtensionType function

Update the extension type used
